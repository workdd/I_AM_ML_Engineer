## About

ML 관련 **수학적 원리와 구현으로** 깊이 있게 정리하고 이해하는 것이 이 프로젝트의 목표입니다.

- 핵심 개념의 직관적 이해 + 수학적 유도
- NumPy/PyTorch Scratch 구현
- 예상 Q&A 포함

## Document Template

각 주제는 다음 구조로 정리됩니다:

1. **한 줄 정의** - 30초 안에 설명 가능한 정의
2. **직관적 이해** - 수식 없이 개념 설명
3. **수학적 원리** - 핵심 수식 유도
4. **Scratch 구현** - NumPy/PyTorch 구현
5. **Q&A** - 예상 질문과 답변
